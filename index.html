Introduction to MODFLOW-OWHM

MODFLOW-OWHM is a specific version of MODFLOW that features a conjunctive use, which is the combined use of groundwater and surface water. MODFLOW-OWHM uses a physically-based supply and demand framework. Users can represent farms, streamflow routing, and riparian evapotranspiration with the Farm Process Package (FMP), which is a unique feature of MODFLOW-OWHM.

Note: This guide only reflects MODFLOW-OWHM
